完全同步的 npm 镜像

#### 如何使用 ``` # 安装 cnpm $ npm install -g cnpm # 或者 alias npm $ alias cnpm="npm --registry=http://registry.cnpmjs.org" # 开始从cnpm 安装依赖 $ cnpm install connect # 如果发现cnpm没有同步到最新版本，可以手动触发同步 $ cnpm sync connect # 不允许直接发布到cnpm $ npm publish name $ cnpm sync name ```
